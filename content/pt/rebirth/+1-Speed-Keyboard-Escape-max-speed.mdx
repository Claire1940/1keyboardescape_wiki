Os renascimentos são a base da progressão a longo prazo. Quando você renasce, seu nível de velocidade atual é resetado, mas você recebe um multiplicador permanente que se aplica a todos os ganhos futuros de velocidade. Esta mecânica é essencial para superar os altos requisitos de nível nas fases posteriores.

Os requisitos de nível para renascer aumentam a cada tier, mas as recompensas de multiplicadores correspondentes crescem exponencialmente. Seguir um cronograma rígido de renascimento garante que você não gaste tempo desnecessário grindando níveis com multiplicadores baixos.
